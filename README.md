# Habit Tracker Dashboard

A full-stack web application for tracking daily habits and building better routines. Built as the final project for Harvard's CS50 course.

## Video Demo


## Description

The Habit Tracker Dashboard is a web-based application that helps users create, track, and maintain daily habits. Users can register for an account, create personalized habits with descriptions and frequency settings, and track their progress through an intuitive dashboard interface.

### Key Features:
- **User Authentication**: Secure registration and login system with password hashing
- **Habit Management**: Create, view, and delete personal habits with customizable colors
- **Daily Tracking**: Toggle habit completion with a single click
- **Streak Tracking**: Automatic calculation of current and longest streaks
- **Progress Visualization**: Color-coded history view showing last 30 days
- **Responsive Design**: Mobile-friendly interface that works on all devices

## Technologies Used

- **Backend**: Python with Flask framework
- **Database**: SQLite with structured schema
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Security**: Werkzeug password hashing, session management
- **AJAX**: Asynchronous updates for smooth user experience
